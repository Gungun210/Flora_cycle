## 🌿 The Problem

Every day, **thousands of temples across India** offer millions of flowers as part of religious rituals. Once the prayers are done, these flowers are discarded — thrown into rivers, landfills, or left to rot on roadsides.

- 🏚️ **800+ million kg** of temple flowers are wasted every year in India
- 🌊 Flower waste dumped in rivers contributes to **water pollution**
- 🦠 Rotting flowers release **harmful gases** and attract disease
- ❌ No organized system exists to **collect or recycle** this waste

The tragedy is that these flowers are not truly waste — they are a **raw material** waiting to be transformed.

---

## 💡 Our Solution — FloraCycle

FloraCycle is a **web-based circular economy platform** that:

- 🕌 Connects **temples** with verified **recycling vendors**
- 📦 Enables temples to **log and submit** their daily flower waste
- 🏭 Helps vendors **track, collect, and process** flower pickups
- ♻️ Transforms discarded blooms into **incense sticks, natural dyes, compost, and handcrafted goods**
- 📊 Provides an **admin dashboard** to monitor the entire ecosystem

*Every petal finds a new purpose. Every temple becomes part of the solution.*

## 🚀 How to Run Locally

**Prerequisites:** Node.js installed, MongoDB running locally or MongoDB Atlas account

**Frontend**
```bash
cd frontend
npm install
npm start
```

**Backend**
```bash
cd backend
npm install
node server.js
```

App will run at `http://localhost:3000`

## 🎯 Why It Matters

Beyond the environment, this platform creates real economic value — for waste collectors, for artisans, for small vendors building a livelihood around something that was previously thrown away. The model is scalable to gurudwaras, mosques, and churches across India.

- ✅ Reduces **river and land pollution** from flower dumping
- ✅ Creates **livelihood opportunities** for waste collectors and artisans
- ✅ Supports India's **Swachh Bharat** and sustainability goals
- ✅ Builds a **circular economy** around religious practices
- ✅ Scalable to **mosques, churches, and gurudwaras** as well
